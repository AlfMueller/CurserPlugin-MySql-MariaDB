# Changelog

## 1.0.0 (2026-02-21)

### Features

- **MCP Server** with 20 tools for MySQL/MariaDB
  - Schema inspection: list databases, tables, columns, indexes, CREATE statements
  - Query execution with EXPLAIN support
  - Data manipulation (insert, update, delete) with safety guards
  - Database objects: views, stored procedures, functions, triggers
  - Server monitoring: process list, status, variables
- **MCP Resources** for passive schema context
- **Connection Pooling** via mysql2 for high performance
- **SQL Safety Rules** preventing accidental destructive operations
- **Setup Skill** for guided installation and configuration
- **Schema Design Skill** with best practices
- **Database Architect Agent** for schema consultation and query optimization
